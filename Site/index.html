About

ErrorSolve is an application created during Canada's Biggest Hackathon, McHacks. It features a python script that is executed while compiling a python program in order to catch any errors and display possible related solutions on your browser. Let us do the work for you!

Hassle Free and Super Easy

Easier than opening a can of soup!

How to use

In order to use our application, download the script from our github. Once downloaded the program can be run by doing the following:

$ ./errorsolve.py python filename.py

Why use this application?

Because we made it in 24 hours! And you will never have to search for solutions manually ever again!
